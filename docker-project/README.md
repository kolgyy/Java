# docker-project

Spring Boot приложение, подготовленное для запуска в Docker.

## Description

Проект демонстрирует контейнеризацию Java-приложения
и его запуск в изолированной среде.

## Purpose

- Освоение Docker
- Контейнеризация backend-приложений
- Подготовка к деплою

## Technologies

- Spring Boot
- Docker
- Maven
