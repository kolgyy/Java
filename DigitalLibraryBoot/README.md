# DigitalLibraryBoot

REST-приложение для управления цифровой библиотекой, построенное на Spring Boot.

## Description

Приложение предоставляет API для работы с библиотечными сущностями
(книги, авторы, пользователи) и демонстрирует построение backend-сервиса
с разделением слоёв (controller, service, repository).

## Purpose

- Реализация REST API
- Работа с базой данных
- Использование Spring Boot как основной платформы

## Technologies

- Spring Boot
- Spring Web
- Spring Data JPA
- Maven

## How to Run

```bash
mvn spring-boot:run
