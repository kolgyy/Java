# SensorAppRESTApiProject3

REST API для обработки и хранения данных с сенсоров.

## Description

Приложение принимает показания сенсоров, валидирует их
и сохраняет в системе.

## Purpose

- Реализация REST-сервиса
- Работа с DTO и валидацией
- Клиент-серверное взаимодействие

## Technologies

- Spring Boot
- Spring Web
- REST API
- Maven
