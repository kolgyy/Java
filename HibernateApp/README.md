# HibernateApp

Приложение для демонстрации работы с базой данных через Hibernate.

## Description

Проект показывает маппинг сущностей, выполнение CRUD-операций
и взаимодействие с БД без использования Spring Boot.

## Purpose

- Освоение Hibernate ORM
- Работа с сущностями и сессиями
- SQL ↔ Java объектное отображение

## Technologies

- Hibernate
- Maven
- JDBC
