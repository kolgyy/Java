# ProductNotifier

Микросервисная система уведомлений о продуктах.

## Description

Проект состоит из нескольких микросервисов:
- сервис управления продуктами
- сервис email-уведомлений

Между компонентами реализовано взаимодействие через события.

## Purpose

- Изучение микросервисной архитектуры
- Асинхронная обработка событий
- Разделение ответственности сервисов

## Technologies

- Spring Boot
- REST
- Maven
