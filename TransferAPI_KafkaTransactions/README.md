# TransferAPI_KafkaTransactions

Система обработки транзакций с использованием Apache Kafka.

## Description

Проект реализует обмен сообщениями между сервисами,
обрабатывающими платежи и депозиты.

## Purpose

- Асинхронные транзакции
- Работа с Kafka
- Event-driven архитектура

## Technologies

- Spring Boot
- Kafka
- REST API
