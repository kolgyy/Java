# SpringSecurityApp

Приложение с реализованной системой аутентификации и авторизации.

## Description

Проект демонстрирует настройку Spring Security,
работу с пользователями, ролями и доступами.

## Purpose

- Изучение Spring Security
- Защита эндпоинтов
- Работа с ролями и логином

## Technologies

- Spring Boot
- Spring Security
- Maven
